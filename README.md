# VR Meteo

Real-time weather forecast web app built with vanilla HTML, CSS, and JavaScript.

## Features

- Search weather by city name
- Current conditions: temperature, humidity, wind speed
- 7-day forecast
- Toggle between °C and °F
- Responsive design

## Tech Stack

- [Open-Meteo API](https://open-meteo.com) — free, no API key required
- Geocoding via Open-Meteo Geocoding API
- Google Fonts: Sora + Fira Mono

## Usage

Just open `index.html` in a browser — no build step, no dependencies.
